Bureau du Fun

A collection of web games to keep distracted during lockdown, December 2020

About

The Bureau du Fun (literally "desktop of fun") is a website I built during lockdown to facilitate socializing virtualy with my friends & family. It's a collection of online multiplayer games which only require an internet browser and a voice chat to play. Visit it here!

Tech details

Made with plain ol' html css JS but then I used this project to experiment with front end tech stacks like gulp, various css / js processors and minifiers, browsersync, parcel and hyperapp.

Project Highlights

  • fully responsive
