MixAR

A hybrid board game that is also a comic book, April 2019

About

MixAR is an interactive comic book where players decide how the story unfolds through their actions. It is a hybrid game that requires both physical cards and a smartphone app to be played.

You can download the Android app and print-to-play kit (in French) or find the code on Github

Tech details

The app was made in Unity C# with Vuforia AR kit. It features a cards detection screen, an Augmented Reality scene, and comic boxes with depth and parallax. We made sure that the smartphone app was a key component of the game without getting cumbersome.
