Beetle

A WebGL generic product configurator, August 2018

About

The beetle configurator is a front-end web app that allow the user to personalise colors and textures of a 3D product as well as print text and logos on a given surface. At heart, it is a generic product configurator which can be quickly cloned and adapted to meet the needs of a client with a tight budget.

Tech details

This app was developed with BabylonJS library, styled with sass and built with NodeJS, npm and grunt. User configurations can be loaded and saved on disk.
